# VideoConvert

Conversor de vídeos implementado com a interface gráfica WPF (Windows Presentation Foundation) e utiliza o utilitário FFmpeg para realizar a conversão dos arquivos.

O programa permite que os usuários selecionem arquivos de vídeo de diversos formatos e os convertam para o formato .mp4. A interface exibe uma lista de vídeos, com um indicador de progresso para cada arquivo, enquanto o processo de conversão é executado de forma assíncrona.

O programa usa o utilitário externo [FFmpeg](https://www.ffmpeg.org/) para realizar a conversão de vídeos.
